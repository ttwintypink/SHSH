# CLAN Event Manager v34

Исправленная стабильная версия:

- `/event create` сразу сохраняет событие в базу как `preparing`.
- Выдача роли `ещё не отметился` идёт отдельной задачей, команда не зависает.
- После выдачи ролей событие переводится в `scheduled`.
- Если время публикации уже наступило, embed публикуется сразу после выдачи ролей.
- `/event list` перезапускает зависшую подготовку, если бот был перезапущен.
- `/event clear_roles` быстрее чистит роли через список участников ролей, а не через полный перебор сервера.
- В логи добавлены подробные строки `[ClanEvent]`.

Для обновления достаточно заменить `SH_discord_bot_split/event_manager.py` и перезапустить бота.
